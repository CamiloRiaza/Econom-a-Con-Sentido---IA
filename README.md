## ◈ El Problema que Resuelvo

> **El 90% del contenido financiero en internet es demasiado complejo o peligrosamente superficial.**
> La clase media recibe "ruido" mediático. Las instituciones reciben "señales" claras.
> *Economía con Sentido cierra esa brecha.*

Un ecosistema de educación financiera multiplataforma — distribuido en **X/Twitter, YouTube y WhatsApp** — que decodifica la macroeconomía, la inflación y las estrategias de protección de riqueza (Bitcoin, Renta Variable) para una audiencia no técnica. Construido y operado en solitario, utilizando la **Inteligencia Artificial como multiplicador de fuerza**.

**Mi rol:** Director de Estrategia · Orquestador de IA · Prompt Engineer

## ◈ Los Dos Agentes de IA

### `AGENTE 01` — El Estratega Macro *(Editor Jefe)*

Una IA con personalidad condicionada *(Custom Persona)* entrenada para pensar como un analista financiero *contrarian*. Su objetivo es filtrar el ruido y encontrar la asimetría del mercado.

| Atributo | Detalle |
|----------|---------|
| **Input** | Reportes financieros crudos, decisiones de la FED, métricas on-chain, geopolítica |
| **Framework** | Fórmula propietaria **M.M.A.** (Macro → Micro → Acción) |
| **Output** | Hilos persuasivos para X · Copys para newsletters · Emisiones de WhatsApp |
| **Voz de Marca** | Cruda · Directa · Empática pero con alta autoridad |
| **Guardrails** | Control de alucinaciones · Cero clichés de IA · Cumplimiento de línea editorial |

**Ejemplo de Prompt Engineering (Condicionamiento de Tarea):**
```
"A partir del siguiente reporte técnico crudo, extrae la 'Tesis Central' y formula
3 'Ganchos de Atención'. Integra los datos duros (como ratios PER o niveles de
soporte del S&P 500) en una narrativa que contraste el pánico mediático con la
oportunidad institucional. Estructura el output en un hilo de X de máximo 7 tweets."
```

### `AGENTE 02` — El Director de Arte *(IA Visual)*

Opera con una Base de Conocimiento *(RAG)* de 4 documentos maestros para garantizar una identidad visual inquebrantable:

```
📁 Base de Conocimiento/
├── 📄 Manual de Identidad Visual    → Códigos HEX · Tipografía · Reglas de Marca
├── 📄 Guía de Ganchos Psicológicos  → Framework de retención de los primeros 3 seg.
├── 📄 Manual de Montaje Híbrido     → Regla de los 4 seg. · Parallax · Dark Nodes
└── 📄 Estructuras de Guion          → Plantillas para Short-form & Long-form
```

**Ejemplo de Prompt Visual (Midjourney / Imagen 3):**
```
"Imagen ultra realista, estilo fotorrealista cinematográfico de alto contraste,

verticalmente creando luz blanca y nítida. En primer plano: manos con traje
```

## ◈ Pipeline de Producción

```
Fase 1  ──────────────────────────────────────  INVESTIGACIÓN
        Ingesta humana de datos: gráficos de TradingView,
        reportes macro (FRED, Truflation), inteligencia
        geopolítica, análisis on-chain.

Fase 2  ──────────────────────────────────────  SÍNTESIS
        El Agente 01 filtra el ruido → extrae la tesis →
        redacta el copy persuasivo usando el framework M.M.A.

Fase 3  ──────────────────────────────────────  DIRECCIÓN DE ARTE
        El Agente 02 genera los prompts exactos de imágenes +
        guiones técnicos (timing y sound design) para los videos.

Fase 4  ──────────────────────────────────────  PRODUCCIÓN
        Human-in-the-loop: edición con keyframes en CapCut,
        diseño de sonido y publicación omnicanal.
```
